Introduction

How accessible is the Rainforest Alliance web page at https://www.rainforest-alliance.org/?

This digest can help answer that question. Nine different tools (Alfa, ASLint, Axe, Equal Access, HTML CodeSniffer, Nu Html Checker, QualWeb, Testaro, and WAVE) tested the page to check its compliance with their accessibility rules. In all, the tools define about 915 rules, which are classified here into about 300 accessibility issues.

The results were interpreted to yield a score, with 0 being ideal. The score for this page was 5185, the sum of 690 for the count of issues, 3612 for specific issues, 769 for tool-by-tool ratings, 0 for the page preventing tools from running, 114 for browser warnings, and 0 for delayed page responses.
