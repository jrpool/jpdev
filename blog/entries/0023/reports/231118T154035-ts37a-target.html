Introduction

How accessible is the Benetech web page at https://benetech.org/?

This digest can help answer that question. Nine different tools (Alfa, ASLint, Axe, Equal Access, HTML CodeSniffer, Nu Html Checker, QualWeb, Testaro, and WAVE) tested the page to check its compliance with their accessibility rules. In all, the tools define about 915 rules, which are classified here into about 300 accessibility issues.

The results were interpreted to yield a score, with 0 being ideal. The score for this page was 2938, the sum of 650 for the count of issues, 1693 for specific issues, 334 for tool-by-tool ratings, 0 for the page preventing tools from running, 248 for browser warnings, and 13 for delayed page responses.
