Introduction

How accessible is the Open Collective Foundation web page at https://opencollective.com/foundation?

This digest can help answer that question. Nine different tools (Alfa, ASLint, Axe, Equal Access, HTML CodeSniffer, Nu Html Checker, QualWeb, Testaro, and WAVE) tested the page to check its compliance with their accessibility rules. In all, the tools define about 915 rules, which are classified here into about 300 accessibility issues.

The results were interpreted to yield a score, with 0 being ideal. The score for this page was 18446, the sum of 800 for the count of issues, 14638 for specific issues, 2956 for tool-by-tool ratings, 0 for the page preventing tools from running, 45 for browser warnings, and 7 for delayed page responses.
