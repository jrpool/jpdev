Web pages for testing

The Autotest project needs exemplar web pages to act as test targets. Some such pages are here.

  • Minimal page designed to pass all packaged and custom Autotest tests, because it has almost no content
  • Passing page designed to pass all packaged and custom Autotest tests, despite containing elements that could fail all custom tests.
  • Failing page designed to contain at least one failure of each custom Autotest test.
