Accessibility of major CVS web pages

Jonathan Pool

Introduction

The following table reports the results of executing version 6 of the a11y procedure of the Autotest application on various web pages of CVS Health.

Findings

The tests include three external packages of tests: Axe, IBM Accessibility Checker, and WAVE. In addition, there are sixteen accessibility tests defined in Autotest, chiefly testing aspects of accessibility that the test packages do not cover.

For each page, Autotest generated a deficit score. A score of 0 indicates that a page has passed all the tests. Any deficit score greater than 0 indicates that at least one test has found at least one fault on the page.

The tests were laslt conducted on 17 November 2021.

In the table below:

  • Each name in the Page column is a link to the page that was tested.
  • Each number in the Deficit column is a link to the detailed results.
