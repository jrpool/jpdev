Accessibility of major CVS web pages

Jonathan Pool

Introduction

The following table reports the results of executing the p03c13 accessibility test battery of the Autotest application on various web pages of CVS Health.

Findings

The tests include three external packages of tests: Axe, IBM Accessibility Checker, and WAVE. In addition, there are thirteen accessibility tests defined in Autotest, chiefly testing aspects of accessibility that the test packages do not cover.

For each page, Autotest generated a deficit score. A score of 0 indicates that a page has passed all the tests. Any deficit score greater than 0 indicates that at least one test has found at least one fault on the page.

The tests were conducted on 4 October 2021.

In the table below:

  • Each name in the Page column is a link to the page that was tested.
  • Each number in the Deficit as a number column is a link to the detailed results.
