Accessibility of major CVS web pages

Jonathan Pool

Introduction

The following table reports the results of executing the Axe, IBM Accessibility Checker, and WAVE accessibility test packages within the p03c13 accessibility test battery of the Autotest application on various web pages of CVS Health.

Findings

For each package and each page, Autotest generated a deficit score. A score of 0 indicates that a page passed all the tests of the package. Any deficit score greater than 0 indicates that at least one test in the package found at least one fault on the page.

The tests were conducted on 4 October 2021.

In the table below, each name in the Page column is a link to the page that was tested.
