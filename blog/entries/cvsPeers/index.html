CVS and competitors: Web home page accessibility

Jonathan Pool

Introduction

These data are results of running procedure a11y03 of Autotest on the home pages of CVS and four competitors.

This procedure performs 421 accessibility tests, some custom-created and most imported from commonly used open-source and academic packages (Axe, WAVE, and Equal Access).

The procedure generates a score for each page. The lower the score, the better. A score of 0 would indicate that a page passed all the tests.

In the table below:

  • Each name in the Page column is a link to the page that was tested.
  • Each number in the Score column is a link to a detailed report.
